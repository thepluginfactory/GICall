Unlocks advanced visibility and styling features for FB Call Now.

== Description ==

FB Call Now Pro unlocks powerful visibility and styling controls in the FB Call Now plugin:

= Pro Features =

* **Day-of-Week Visibility** - Show your button only on specific days of the week
* **Time Window Controls** - Set business hours for button visibility, with support for overnight schedules
* **Device Targeting** - Show or hide the button on desktop, tablet, or mobile devices
* **Custom Fonts** - Choose from 100+ Google Fonts or use your Divi theme fonts
* **Debug Logging** - Troubleshoot visibility issues with detailed logs

= Requirements =

This plugin requires [FB Call Now](https://wordpress.org/plugins/fb-call-now/) (Free) version 4.1.77 or higher to be installed and activated.

== Installation ==

1. Make sure FB Call Now (Free) is installed and activated
2. Upload the `fb-call-now-pro` folder to `/wp-content/plugins/`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to FB Call Now > Pro Settings to configure the Pro features

== Frequently Asked Questions ==

= Does this plugin work without the free version? =

No, FB Call Now Pro requires the free version of FB Call Now to be installed and active. It unlocks additional features within the free version.

= How do I set up business hours? =

Go to FB Call Now > Pro Settings and configure the Time Window section. Set your start and end times, and enable "Wrap to Next Day" if your business hours extend past midnight.

= Can I show different buttons on mobile vs desktop? =

Yes! Use the Device Visibility section to choose which device types should display your buttons.

= How do I use a custom font? =

In Pro Settings, scroll to Font Family and select "Google Font". Then search for and select your preferred font from our curated list of 100+ popular Google Fonts.
